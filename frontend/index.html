Shell Profiles

Saved shells shown in the selection screen.

Bell Actions

Actions to perform when the terminal bell (BEL) is triggered. Multiple actions can run simultaneously.

Terminal Key Bindings

Keys that don't match any binding are passed through to PTY (Ctrl+C → SIGINT, Ctrl+D → EOF, etc.). Guards: &selected / &not-selected.

Command Mode

Press the prefix key in the terminal to enter command mode. Then press J to focus the input box.

0 = no timeout (stay in command mode until explicitly cancelled)

Focus Actions

Actions to perform when focus changes between the terminal and input box.

Turn IME off when terminal is focused, on when input box is focused

Input Box Key Bindings

Each key combination can be assigned an action: push, push-and-follow, execute, execute-and-follow, newline, or none.
